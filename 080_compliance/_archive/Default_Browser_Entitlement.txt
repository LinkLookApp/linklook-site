LINKLOOK — DEFAULT BROWSER ENTITLEMENT
Version: 1.0
Date: 2026-03-07

LinkLook is designed to function as the default browser (entitlement status: to be
requested) so it can receive URLs opened from other apps
and evaluate them for phishing and scam risks before navigation occurs.
If safe, the URL opens in LinkLook's built-in browser (WKWebView).
LinkLook IS the user's primary browser — there is no handoff to an external
browser. See CLAUDE.md rule 10.
