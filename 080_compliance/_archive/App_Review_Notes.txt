================================================================================
APP REVIEW NOTE (paste into App Store Connect)
================================================================================

LinkLook is a safety-driven browser that checks links before they open.

How it works:
When a user taps a link in Messages, Mail, or any other app, iOS delivers the
URL to LinkLook (designed to function as the default browser; entitlement
status: to be requested). LinkLook runs a safety
check BEFORE loading the page and then routes by verdict:
- OK: opens immediately in LinkLook with a brief non-blocking confirmation toast
- INFORM: shows a light warning interstitial with Go Back and Continue
- WARN: shows a stronger warning interstitial with Go Back and Continue Anyway
- BLOCK: shows a blocking warning with no forward path

The safety check combines:
- Structural URL analysis (domain patterns, brand lookalike detection, TLD
  category, IP address detection, URL shortener detection)
- Google Safe Browsing API (hash-prefix lookup for known malware, phishing,
  social engineering, and unwanted software)

All normal browsing happens inside LinkLook — LinkLook is the user's primary
browser. LinkLook does not present a user-facing "Open in other browser"
option. LinkLook never loads a page before the relevant allowed state is
reached.

Entry points:
- Omnibox (type or paste a URL)
- Default browser handoff (links from other apps)
- QR code scanner (built-in camera scanner)
- Share Sheet ("Check with LinkLook" from any app)

LinkLook does not act as a VPN or proxy. It evaluates URLs locally on-device
and sends only SHA-256 hash prefixes to Google Safe Browsing (no full URLs).
A local check history is stored on-device and can be cleared or disabled by
the user at any time.

Test instructions:
1. Open the app and type "apple.com" — the page should open in LinkLook and show a brief OK toast
2. Type "bit.ly/test" — should show INFORM or WARN depending on current policy
3. Tap "Go Back" — returns to start page
4. From an INFORM screen, tap "Continue →" — opens in LinkLook's built-in browser
5. Home button returns to start page from any state

No login required. No demo account needed.

================================================================================
PRIVACY NOTES
================================================================================

Data sent off-device:
- Google Safe Browsing: SHA-256 hash prefixes of URLs (not full URLs).
  Used to check against Google's threat database. Required for phishing
  and malware detection.

Data stored on-device only:
- Check history: URL, verdict, reason code, timestamp, entry source.
  User can clear or disable at any time. Never leaves the device.
- Feature vector log: numeric URL features (no raw URLs) for future
  on-device AI model training. Never leaves the device.

No personal identifiers collected. No analytics SDK. No advertising SDK.
No third-party tracking.
